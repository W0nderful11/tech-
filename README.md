# Tech Project

This repository contains AI-powered applications built with Streamlit and Google Gemini.

## Files

- `agentic_ai_assist.py`: AgenticAI Assist - Task management with AI agents
- `research_agent.py`: Research Agent - Automated research reports
- `requirements.txt`: Python dependencies
- `.env`: Environment variables (add your GOOGLE_API_KEY)

## Setup

1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Google Gemini API key to `.env`
4. Run: `streamlit run agentic_ai_assist.py` or `streamlit run research_agent.py`

## Deploy

- **Streamlit Cloud**: Upload to https://share.streamlit.io/
- **Local**: Run with `streamlit run`

## Features

- User-Verified Autonomy for tasks
- AI chat with Gemini
- Research automation
- Task lifecycle management
